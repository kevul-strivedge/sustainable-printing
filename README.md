# Sustainable Printing Platform

Modern printing ecommerce platform built with Next.js, TypeScript, and Tailwind CSS.

---

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Zustand
- Axios
- React Hook Form

---

## Features

- SEO Optimized
- Dynamic Product Configuration
- Real-time Pricing
- Pincode Delivery Calculation
- File Upload System
- Multi-step Checkout
- Payment Integration
- Responsive Design

---

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```

---
